## Debug

1. npm run nexus
1. react-native start
1. attach packager to debugger via Visual Code Source
1. react-native run-android
1. active remote js debugger and hot reload via Ctrl^M on emulator

## Start via Expo

1. npm install -g exp
1. exp start (--clear ro reset cache)
1. scan barcode
